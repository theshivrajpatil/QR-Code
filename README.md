# QR Code Generator

A simple and user-friendly QR Code Generator application built using Python.
This tool allows you to quickly generate QR codes for any text or URL and save them as images.

# Features

1. Generate QR codes from text or URLs
2. Save QR code images locally
3. Lightweight & fast
4. Beginner-friendly project structure
5. Open-source and free to use

# Tech Stack
	•	Python 3.x
	•	qrcode library
	•	Pillow (PIL)
